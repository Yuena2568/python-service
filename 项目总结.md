## 🚀 启动项目（一键流程）

### 第一次启动
```bash
# 1. 进入项目目录
cd d:\新建文件夹\python-service

# 2. 激活虚拟环境
python-service\Scripts\activate

# 3. 启动API服务
python -m app.main
```
